evrptwv2g.utils.utilities.calculate_distance_matrix

evrptwv2g.utils.utilities.calculate_distance_matrix(df: DataFrame, metric: str = 'euclidean', dist_type='scipy', instance: str = '', save: bool = False) → DataFrame

Calculates a square distance matrix given a set of vertices with (x,y) coordinates.

Parameters

  • df (np.array) – Vx2 matrix for V vertices with (x,y) coordinates

  • metric (str) – scipy distance metric, default is ‘euclidean’

Returns

Square matrix of distances between every vertex.

Return type

pd.DataFrame
